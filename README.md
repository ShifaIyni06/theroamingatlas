# 🌍 THE ROAMING ATLAS

A premium travel website built to explore curated travel packages across India with a seamless and modern user experience.

---

## ✨ Live Preview

🔗 https://theroamingatlas.vercel.app/

---

## 🚀 Features

* 🏔️ **Curated Travel Packages** – Explore hand-picked destinations
* 🎨 **Modern UI/UX** – Clean, premium design with smooth animations
* 📸 **Masonry Gallery** – Beautiful travel memories in grid layout
* 💬 **Direct Booking via WhatsApp** – Instant communication
* 📧 **Mail & 📞 Call Integration** – Easy contact options
* 🌗 **Dark/Light Mode Toggle**
* 📱 **Fully Responsive Design**

---

## 🛠️ Tech Stack

* ⚛️ React + TypeScript
* 🎨 Tailwind CSS
* 🎞️ Framer Motion
* ⚡ Vite
